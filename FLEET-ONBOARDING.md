## Communication
- A2A protocol: FLUX opcodes 0x60-0x6B
- Async: git commits (I2I protocol)
- Sync: Lighthouse (Oracle1) for fleet coordination

## Trust
- Start with TRUST_SEED (0xCD) for new vessels
- Build trust through reliable task completion
- Trust has 5 dimensions: competence, integrity, benevolence, identity, longevity

## Energy
- ATP is the compute currency
- Contribute to fleet pool in good times
- Request from fleet pool in bad times
- Circadian modulation affects generation rate
